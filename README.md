# SkillHub Academy Platform

## 📖 Loyiha tavsifi

**SkillHub Academy** — bu NestJS va HBS (Handlebars) yordamida yaratilgan mini Learning Management System (LMS) hisoblanadi.

Tizim quyidagi imkoniyatlarni taqdim etadi:

- Talabalar ro‘yxatdan o‘tishi va kurslarga yozilishi
- O‘qituvchilar kurslar yaratishi
- Topshiriqlar yaratish va boshqarish
- Topshiriqlar uchun fayllarni yuklash
- Telegram orqali xabarnomalar yuborish
- Email manzilini tasdiqlash (Email Verification)
- Frontend qismini NestJS ichida HBS (Handlebars) yordamida render qilish

---

## 🚀 Loyihani ishga tushirish

Loyihani ishga tushirishdan oldin `.env` faylini kerakli konfiguratsiyalar bilan to‘ldiring.

So‘ng quyidagi buyruq orqali loyihani ishga tushiring:

```bash
pnpm start:dev
```
